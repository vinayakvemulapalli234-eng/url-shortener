# ⚡ SnapURL — URL Shortener

A full-featured URL shortener built with Node.js, Express, and Supabase.

🔗 Live Demo: https://snap-url-w0o8.onrender.com

## Features
- 🔐 User authentication (register/login)
- ✂️ Shorten any URL instantly
- 📊 Click tracking & analytics
- 📋 Copy to clipboard
- 🌙 Beautiful dark mode UI

## Tech Stack
- **Backend:** Node.js, Express
- **Database & Auth:** Supabase (PostgreSQL)
- **Frontend:** Vanilla HTML, CSS, JavaScript
- **Hosting:** Render.com

## Setup Locally
1. Clone the repo
   git clone https://github.com/vinayakvemulapalli234-eng/url-shortener.git
2. Install dependencies
   npm install
3. Create .env file
   SUPABASE_URL=your-supabase-url
   SUPABASE_ANON_KEY=your-anon-key
   PORT=3000
4. Run the server
   node server/index.js
5. Open http://localhost:3000
